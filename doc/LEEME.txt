(I) Introducción
--------------------------------------------
GR-lida es un multi-lanzador con el que podrás manejar de forma sencilla
tres emuladores que se usan principalmente para el mundo abandonware.

 - DOSBox emulador de MS-DOS con Sound Blaster o Gravis Ultrasound, principalmente
     para juegos basados en 486.
 - ScummVM es un emulador o interprete principalmente para los juegos de Lucar-Arts 
     y otras compañías como Revolution Software.
 - VDMSound principalmente es un emulador de sonido para windows NT, 2000 y XP


(II) Compilar el programa
--------------------------------------------
Vea el fichero: INSTALL.txt


(III) Otras características
--------------------------------------------
 - Actuar como Base de Datos de Juegos:
   En ella podrás añadir los juegos y poner a que genero pertenece, en que año se publicó,
   la compañía que lo creó, comentario del juego entre otras cosas.
 - Importar datos y configuraciones de los juegos desde una web preparada para ello.
 - Soporte para mostrar smilies en los comentarios de los juegos
 - Añadir Iconos para cada Juego.
   Con esta función podrás añadirle un icono predeterminado o uno propio para identificar
   cada juego en la lista de GR-lida, soportando los siguientes formatos: *.png; *.gif;
   *.jpg; *.bmp; de un tamaño máximo de 24x24 píxeles.
 - Comentario del Juego con soporte de HTML
   En el comentario se le puede añadir código html para darle la apariencia que desees,
   para ver las etiquetas que soporta mira la documentación de las QT
   http://doc.trolltech.com/4.2/richtext-html-subset.html
